## 课程简介

CS50 是哈佛大学著名的计算机科学入门课程，涵盖了计算机科学和程序设计的基础知识。课程内容由浅入深，非常适合零基础和有一定基础的学习者。

## 周别内容

### 第 0 周：计算机科学导论

- **概述**：计算机科学是什么，计算机如何工作
- **重点**：
  - 二进制和数据表示
  - 高级和低级编程语言
  - Scratch 编程入门

### 第 1 周：C 语言

- **概述**：了解 C 语言的基本语法和结构
- **重点**：
  - 基本数据类型
  - 变量和常量
  - 条件语句和循环
  - 简单的输入输出

### 第 2 周：数组

- **概述**：数组在编程中的应用
- **重点**：
  - 定义和使用数组
  - 字符串处理
  - 简单的数组操作

### 第 3 周：算法

- **概述**：算法的基本概念和常见算法
- **重点**：
  - 线性搜索和二分搜索
  - 排序算法（冒泡排序、选择排序、插入排序）

### 第 4 周：内存

- **概述**：内存管理和指针的使用
- **重点**：
  - 指针基础
  - 动态内存分配
  - 内存泄露和调试
